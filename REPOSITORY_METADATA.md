## Description

This release contains the frozen code, configuration files, figure source data, derived tables, literature-screening records and quality-control evidence for a methodological audit of radioiodine-related transcriptomic programs in thyroid cancer. The audit separates public recoverability, clinical endpoint alignment, patient-deletion stability, exact-label and covariance-matched program-identity calibration, sensitivity to deterministic cohort collinearity and donor-aware cellular-compartment preference. It does not provide a clinical diagnostic classifier.
